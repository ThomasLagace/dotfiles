Dotfiles
========
My Linux desktop configuration. It is intended to be purely functional (eyecandy sold separately).
*New: eyecandy now included!*

- i3 
- polybar
- nvim config
- zshrc 
- ncmpcpp
- termite (this terminal is no longer used by me)
- alacritty (much cooler terminal)
